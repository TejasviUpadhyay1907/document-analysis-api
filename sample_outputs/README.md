# Sample API Outputs

This folder contains real API response samples from testing the document analysis endpoint with various document types.

| File | Document Type | Description |
|------|--------------|-------------|
| `article_output.json` | article | Technology Industry Analysis PDF |
| `cover_letter_output.json` | official_letter | Cover letter image (PNG) |
| `epa_letter_output.json` | official_letter | EPA government letter PDF |
| `id_card_output.json` | identity | Student ID card image |
| `notice_output.json` | notice | College internship notice PDF |
| `resume_pdf_output.json` | resume | Resume PDF |

These outputs demonstrate the API's ability to correctly classify document types, extract entities, generate summaries, and classify sentiment across diverse real-world documents.
